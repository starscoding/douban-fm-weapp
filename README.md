# 豆瓣 FM (第三方)微信小程序客户端

## 环境要求

由于使用 webpack 2.0, 需要 `Node.js` 5.0 以上

## 运行

```js
npm run start
```

微信开发者工具中项目路径指向 `dist` 目录即可
